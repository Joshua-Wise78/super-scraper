# WiseBot

 **Your friendly, "not-so-smart" Discord bot built in Python.**

## Features

### 1. Wiki & Knowledge Retrieval (WIP)
*Automated fetching of information from game Wikis and Fandom pages.*

* **JSON Data Management:** Store, retrieve, and list Wiki sites using local JSON storage.
* **Fandom Sub-domain Scraper:**
    * Capable of grabbing sub-domains from larger fandom pages.
    * *Example:* Uses `escapefromtarkov` as the main domain to search for specific items like `Salewa`.

### 2. Server API Management (WIP)

**Immich (Photo Management)**
* **Quick Search:** Find photos and files inside your Immich instance directly from Discord.
* **Upload:** Quickly add photos or files to your Immich library.
* **Monitoring:** Receive alerts for storage usage and backup status.
* **Daily Fun:** "Random Photo of the Day" feature.

**Crafty Controller (Minecraft)**
* **Status Checks:** View the live status of your Minecraft servers.
* **Power Control:** Stop and Run different server instances remotely.

### 3. Media Controls

**Jellyfin**
* **Download Listener:** Sends notifications when movies and shows finish downloading.
* **Session Manager:** View and manage active playback sessions.

**Jellyseerr**
* **Requests:** Manage and view requests for new movies and TV shows.
